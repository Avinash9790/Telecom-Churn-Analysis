# Telecom Churn Analysis 🚀

**End-to-end SQL analysis** identifying high-risk customer segments across **7,043 telecom customers** (26.54% churn rate).

## 📊 Business Problem
**26.54% churn rate** costing $2,279 lifetime value per lost customer. 
Goal: Identify patterns driving churn and actionable retention strategies.

## 🔥 Key Findings (20 SQL Queries)
| # | Insight | Churn Rate |
|---|---------|------------|
| 1 | **Senior + Month-to-month + Fiber** | **57.86%** |
| 2 | Month-to-month contracts | **42.71%** |
| 3 | Fiber optic customers | **41.89%** |
| 4 | Senior citizens | **41.68%** |
| 5 | Electronic check payments | **~45%** |

**Month-to-month Fiber customers churn 7x more than 2-year contracts.**

## 🛠 Tech Stack
SQL (MySQL) → 20 advanced queries
Python (EDA) → Power BI (dashboards)
Dataset: IBM Telco Churn (Kaggle)



## 🎯 Business Impact
- **Identified top 5 churn drivers** across 20 factors
- **Revenue protection**: Each retained customer = $2,279 LTV  
- **Action plan**: Convert month-to-month → annual contracts
